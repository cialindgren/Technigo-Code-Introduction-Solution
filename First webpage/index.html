HTTPS - adding security

What is HTTPS?
    • HTTPS stands for Hypertext Transfer Protocol Secure
    • It adds another security layer on data in transit
    • It enables safe a connection and communication

    HTTPS provides an additional security level to the HTTP protocol, which on itâ€™s own is unsecured. HTTP transfers data from user to web server unsecured, making it attainable to second party users. HTTPS main use is therefore to encrypt data (e.g. credit card and billing details, user login etc.) to prohibit hackers or other intermediate attackers from attaining sensitive information from the data in transit between user and web server.
